# TragetExplorer

Explore the subdomain of target website

## Installation

```
chmod +x start.sh 
```
## How to use tool

1. ```bash start.sh```
1. Enter the target name (For Example: google.com) without http or https

## Requirement Tools 

1. subfinder
1. amass
1. httpx
1. uniq
